# Marketing Intelligence Report Spec

## Monitoring Brief Schema

Use this compact brief before researching:

```text
Brand:
Market:
Period:
Product lines:
Competitors:
Owned accounts:
Platforms:
Visual inspiration sources:
Case owner needs:
Output language:
Output files:
Caveats:
```

## Evidence Table Fields

Use these columns for competitor and market items:

```text
Source type | Platform | Account / page | Date | What happened | Link | Visual | Evidence quality | Insight | Brand action
```

Evidence quality:

- High: original post/page or official source
- Medium: platform search result, creator page, marketplace listing, or reputable media report
- Low: search snippet only or inaccessible platform preview

## Case Line Fields

```text
Priority | Case direction | Product fit | Materials | User scene | Visual hook | Source/reference | Why it matters | Social packaging
```

Priority:

- P0: shoot or brief immediately
- P1: next batch
- P2: archive for seasonal or campaign use

## Visual Inspiration Fields

```text
Image | Source link | Category | Visual takeaway | Adaptation idea | Matching platform
```

Visual sources can include Pinterest, Etsy, Instagram, Amazon, competitor galleries, craft-market pages, wedding/gift sites, design blogs, or user-generated marketplaces.

## Recommended Source Mix

For a normal weekly report, aim for:

- 3-5 competitor source links
- 2-4 trend or market source links
- 4-8 visual inspiration links
- 4-8 case ideas
- 3-5 action recommendations

## Quality Checklist

Before finalizing:

- Each key fact has a source link.
- The leader version is meaningfully shorter than the full version.
- Images render in a browser or have a clear fallback link.
- Login-wall limitations are stated.
- Case recommendations are actionable for a case owner.
- Visual references include adaptation ideas, not just links.
- The report avoids copying long source text.
- File names match the user's preferred convention.
